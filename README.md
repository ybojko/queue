# Черга прання в гуртожитку

Простий додаток для електронної черги прання з підтримкою 4-го та 6-го поверху.

## Що зробити тобі (Supabase)

**Я не можу створити Supabase проєкт за тебе** — це потрібно зробити на сторонньому сервісі:

1. Зайди на [supabase.com](https://supabase.com) та створи безкоштовний акаунт.
2. Створи новий проєкт (New Project).
3. У проєкті відкрий **SQL Editor** та виконай SQL з `supabase/schema.sql`. Якщо таблиця вже є — виконай лише `supabase/migrations/001_add_session_id.sql`.
4. Перейди в **Settings → API** — скопіюй:
   - **Project URL**
   - **anon public** key
5. У корені проєкту створи файл `.env` (скопіюй з `.env.example`):
   ```
   VITE_SUPABASE_URL=твій_project_url
   VITE_SUPABASE_ANON_KEY=твій_anon_key
   ```

## Запуск

```bash
npm install
npm run dev
```

## Маршрути

- `/` — головна сторінка з чергою
- `/adm-q7x2k9m4p1` — секретна адмін-панель (додавання/редагування записів)

## Функції

- Календар для перегляду різних днів
- Перемикання між 4-м та 6-м поверхом
- **Додавання в чергу** — будь-хто може додати себе (телеграм тег, кімната)
- **Редагування статусу** — кожен може змінювати статус лише своїх записів (сесія зберігається в браузері)
- Таблиця: №, телеграм тег, кімната, статус (очікую / в процесі / закінчив)
- Адмін-панель: додавання записів, зміна статусу, видалення будь-яких записів
